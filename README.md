# JDBC CRUD Operations

A simple Java project demonstrating CRUD (Create, Read, Update, Delete) operations using **JDBC** and **MySQL**.

## 🚀 Features
- Insert records
- Display records
- Update records
- Delete records
- PreparedStatement example
- Transaction Handling

## 🛠️ Tech Stack
- Java
- JDBC
- MySQL

## 📂 Files
- `Insert.java` – Insert data
- `Show.java` – Display data
- `Update.java` – Update data
- `Delete.java` – Delete data
- `PrepStat.java` – PreparedStatement example
- `TransHandling.java` – Transaction handling
- `BP.java` – Database connection

## ▶️ How to Run
1. Configure MySQL database credentials.
2. Add the MySQL JDBC Driver.
3. Run the required Java file for the desired operation.
